# Leap Year
Checks whether the input year is a leap year

Mathematical equations behind calculating a leap year:
- The number to be a modules of 4, 400, or 100
- x % 4
- x % 400
- x % 100


Program Features:
- User enters a year
- Program checks if the given year is a leap year
- Returns output
